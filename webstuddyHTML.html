อธิบาย ภาษา HTML

ภาษา HTML (HyperText Markup Language) เป็นภาษาที่ใช้ในการสร้างและออกแบบหน้าเว็บ

โครงสร้างภาษา HTML

HTML ประกอบด้วยแท็ก (tags) ที่ใช้ในการกำหนดโครงสร้างและรูปแบบของเนื้อหาในหน้าเว็บ

ตัวอย่างโครงสร้าง HTML พื้นฐาน

ประกอบด้วยแท็กหลัก ดังนี้

1.<!doctype html>
คือการประกาศประเภทเอกสารว่าเป็นการเขียนโปรแกรม HTML5
2.<html>
</html>
คือแท็กหลักที่ใช้ในการเริ่มต้นและสิ้นสุดเอกสาร HTML ทุก Code ทุก tag จะอยู่ภายในแท็กนี้
3.<head>
</head>
คือ ส่วนที่ใช้อธิบายข้อมูลเกี่ยวกับ เวป เช่น ชื่อเรื่อง (title), คำอธิบาย (description), คำค้นหาจากผู้ใช้งาน (keywords)
4.<meta...คือ การประกาศรูปแบบต่างๆ ของเว็บนี้>
<meta charset="utf-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<meta name="description" content="เว็บการเรียนรู้ ภาษา HTML">
<meta name="keywords" content="HTML, ภาษา HTML, เรียนรู้ HTML, เว็บ HTML,เขียนเว็บ HTML">
<meta name="author" content="Piyabut Hongtawekul">
4.<body>
</body>
คือ ส่วนที่ใช้ในการแสดงเนื้อหาหลักของหน้าเว็บ เช่น ข้อความ, รูปภาพ, ลิงก์, ตาราง เป็นต้น 
ตัวอย่างที่ 1
       1. <!doctype html>  
       2. <html> คือ แท็กหลักที่ใช้ในการเริ่มต้นเอกสาร HTML โดยจะมี tag เปิด บรรทัดที่2 และ tag ปิด </html> ที่ใช้ในการสิ้นสุดเอกสาร ในตัวอย่างนี้คือ บรรทัดที่ 10
       3. <head>
       4.     <title>My Website</title>
       5. </head>
       6. <body>
       7. <h1>Welcome to My Website</h1>
       8.     <p>This is a sample paragraph.</p>
       9. </body>
       10.</html>

มี Element หลักๆ ดังนี้

  • Heading <h1>, <h2>, <h3>, ...
  • Paragraph <p>
  • Link <a href="www.google.com">...</a>
  • Link <a href="www.google.com" target="_blank">...</a>
  • page2.html
  • page3.html
  • Image <img>
  • List <ul>, <ol>, <li>
